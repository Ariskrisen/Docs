EasyScript 🚀

EasyScript — это легковесный и высокопроизводительный скриптовый движок для серверов Minecraft (Paper 1.21+), созданный как современная и гибкая альтернатива классическому плагину Skript. Вместо кастомного синтаксиса он использует стандартный JavaScript (через GraalVM), позволяя вам использовать всю мощь программирования с прямым доступом к Bukkit/Paper API.

✨ Особенности​

  • Современный JavaScript: Пишите логику на JS (ES6+) с полным доступом к Java-классами и Bukkit API.
  • Интеграция с PlaceholderAPI: Встроенная поддержка плейсхолдеров PAPI.
  • Динамические Команды: Регистрируйте кастомные команды прямо из скриптов.
  • Мощное API Событий: Легко слушайте любые события (Events) сервера.
  • Встроенный Планировщик: Упрощенное управление синхронными, асинхронными и отложенными задачами (Scheduler).
  • Горячая Перезагрузка: Изменяйте скрипты и применяйте обновления мгновенно через /es reload.
  • Быстро и Легко: Минимальная нагрузка на сервер в сравнении с громоздкими скриптовыми языками.

🛠️ Установка​

  1. Скачайте последний EasyScript.jar.
  2. Поместите его в папку plugins вашего сервера.
  3. (Опционально) Убедитесь, что установлены PlaceholderAPI и Vault для полной поддержки.
  4. Перезапустите сервер.
  5. Создавайте свои .js скрипты в папке plugins/EasyScript/scripts/.

📜 Руководство по Скриптингу​

Базовая Структура​

Скрипты располагаются в plugins/EasyScript/scripts/. Каждый файл изолирован и исполняется в собственном контексте (context).

Обработчики событий (Events)​

const PlayerJoinEvent = Java.type('org.bukkit.event.player.PlayerJoinEvent');

events.on(PlayerJoinEvent, (event) => {
    const player = event.getPlayer();
    player.sendMessage("§aДобро пожаловать на сервер!");
});

Регистрация команд​

commands.register("hello", "Команда приветствия", "/hello", (sender, args) => {
    sender.sendMessage("§bПривет от EasyScript!");
});

Использование плейсхолдеров​

const msg = placeholder.setPlaceholders(player, "Ваш баланс: %vault_eco_balance%");
player.sendMessage(msg);

Использование планировщика (Scheduler)​

scheduler.runLater(200, () => { // Выполнится через 10 секунд
    player.sendMessage("Всплывающее сообщение с задержкой!");
});

🎮 Встроенный пример: Улучшение границы мира​

EasyScript идет в комплекте со скриптом border_logic.js, который реализует:

  • Обнаружение первого использования команды /ws home.
  • 10-секундная задержка уведомления.
  • 6-уровневая система прокачки размера границы мира (World Border) через /upgrade-border.
  • Сохранение данных в player_data.json.

💻 Команды​

  • /easyscript reload (Алиас: /es reload) - Перезагружает все скрипты из папки scripts/.
  • /upgrade-border - Пример команды из базового скрипта-примера.

🏗️ Для разработчиков​

Требования​

  • JDK 21
  • Maven

Сборка​

mvn clean package

Готовый JAR файл появится в папке target/ (используйте -shaded.jar для сервера).
