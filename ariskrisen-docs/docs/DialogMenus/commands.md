# Команды и Права (Commands & Permissions)

Управление плагином DialogMenus осуществляется через главную команду `/dialogmenus`.

## ⌨️ Список команд

| Команда | Алиасы | Описание | Право (Permission) |
| :--- | :--- | :--- | :--- |
| `/dialogs reload` | `/dim reload`| Перезагружает все меню из папки `menus/`. | `dialogmenus.reload` |
| `/dialogs open <игрок> <меню>` | `/dim open` | Открывает указанный диалог для игрока. | `dialogmenus.open` |

### Примеры использования:
- `/dim reload` — обновить конфиги после редактирования.
- `/dim open @s main_menu` — открыть себе главное меню.
- `/dim open Player1 shop` — открыть магазин игроку Player1.

---

## 🔐 Права (Permissions)

| Узел (Node) | Описание | По умолчанию |
| :--- | :--- | :--- |
| `dialogmenus.admin` | Полный доступ ко всем функциям плагина. | OP |
| `dialogmenus.reload` | Разрешает использовать команду перезагрузки. | OP |
| `dialogmenus.open` | Разрешает открывать меню (себе и другим). | OP |

> [!TIP]
> Если вы хотите дать игрокам возможность открывать определенные меню самостоятельно (например, через NPC), рекомендуется использовать плагины на NPC или скрипты, которые будут выполнять команду `/dim open %player% <menu>` от имени консоли.
