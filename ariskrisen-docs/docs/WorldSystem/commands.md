# Команды и Права ⌨️

Управление плагином WorldSystem осуществляется через главную команду `/ws`.

## 👤 Команды игрока

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/ws gui` | Открыть главное меню управления миром. | `ws.build` |
| `/ws get [шаблон]` | Создать или получить свой мир. | `ws.get` |
| `/ws home` | Телепортироваться в свой мир. | `ws.tp` |
| `/ws leave` | Покинуть личный мир (возврат на спавн). | (все) |
| `/ws info` | Показать информацию о текущем мире. | (все) |
| `/ws addmember <игрок>` | Добавить участника в ваш мир. | `ws.build` |
| `/ws delmember <игрок>` | Удалить участника из вашего мира. | `ws.build` |
| `/ws delete` | Полностью удалить свой мир. | `ws.delete` |
| `/ws tnt [true/false]` | Включить/выключить урон от TNT на вашем мире. | `ws.build` |
| `/ws fire [true/false]` | Включить/выключить урон от огня на вашем мире. | `ws.build` |
| `/ws players` | Открыть меню управления участниками мира. | (все) |
| `/ws sethome` | Установить точку дома в вашем мире. | `ws.sethome` |

## 🛠 Команды администратора

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/ws tp <мир>` | Телепорт в любой мир игрока. | `ws.tp.other` |
| `/ws get <игрок> <шаблон>` | Создать мир для конкретного игрока. | `ws.get.admin` |
| `/ws reset` | Полностью сбросить базу данных плагина. | `ws.*` |
| `/ws togglebuild` | Включить/выключить режим строительства. | `ws.*` |

---

### Дополнительные права
- `ws.template.<name>` — право на использование конкретного шаблона мира.
- `ws.*` — полный доступ ко всем функциям плагина.
