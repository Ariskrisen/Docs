# WorldSystem 🌍

:::info WorldSystem-Melm является модифицированной и улучшенной версией оригинального плагина WorldSystem. :::

**WorldSystem** — это мощный плагин для управления мирами, где каждый игрок может получить свой собственный, полностью настраиваемый мир. Это идеальное решение для анархии, выживания или творческих серверов, где игрокам нужно личное пространство.

## ✨ Основные возможности
- **Личные миры** - Каждый игрок получает свой приватный мир.
- **Шаблоны миров** - Выбор из различных типов генерации (Plain, Meadows, Desert, Snow).
- **GUI Управление** - Удобный интерфейс для настройки прав, мобов, TNT и многого другого.
- **Оптимизация** - Поддержка асинхронной загрузки и интеграция с Chunky.
- **Экономика** - Интеграция с Vault для платного создания миров.

## 📥 Установка
1. Скачайте последнюю версию c [GitHub Releases](https://github.com/Ariskrisen/WorldSystem-Melm/releases).
2. Поместите JAR-файл в папку `plugins/`.
3. Перезапустите сервер.
4. Настройте шаблоны и права в `config.yml`.

### Зависимости
- **WorldEdit** (обязательно) — для работы с регионами и мирами.
- **Vault** (опционально) — для платного создания миров.
- **PlaceholderAPI** (опционально) — для вывода статистики.
