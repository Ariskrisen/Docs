# Установка и настройка

## Требования

| Требование | Версия | Обязательно |
|------------|--------|-------------|
| Paper/Spigot | 1.21+ | ✅ Да |
| ItemsAdder | 3.x | ❌ Нет (для кастомных текстур) |

## Установка

1. **Скачайте** последнюю версию `ArtifactPlugin-X.X.X.jar`
2. **Положите** в папку `plugins/` вашего сервера
3. **Перезапустите** сервер

После первого запуска создастся файл:
```
plugins/
└── ArtifactPlugin/
    └── config.yml
```

## Структура проекта

```
plugins/ArtifactPlugin/
└── config.yml          # Все артефакты и настройки
```

## Команды

| Команда | Описание | Права |
|---------|---------|--------|
| `/artifact give <id> [игрок] [кол-во]` | Выдать артефакт | `artifact.give` |
| `/artifact list` | Список всех артефактов | всем |
| `/artifact info <id>` | Информация об артефакте | всем |
| `/artifact bag <тир> [игрок]` | Выдать аномальный мешок | `artifact.give` |
| `/artifact reload` | Перезагрузить конфиг | `artifact.reload` |

### Примеры команд

```bash
# Выдать артефакт игроку
/artifact give speed_charm PlayerName

# Выдать 5 артефактов себе
/artifact give speed_charm 5

# Выдать мешок III тира
/artifact bag 3

# Выдать мешок другому игроку
/artifact bag 4 PlayerName
```

## Права доступа

```yaml
permissions:
  artifact.give:
    description: Выдача артефактов
    default: op
  artifact.list:
    description: Просмотр списка артефактов
    default: true
  artifact.info:
    description: Информация об артефакте
    default: true
  artifact.reload:
    description: Перезагрузка конфига
    default: op
```

## Аномальные Мешки

### Что это?

Аномальные мешки — специальные сумки (BUNDLE), которые могут хранить артефакты внутри себя.

### Как получить

```bash
/artifact bag 1    # Мешок I тира (2 слота)
/artifact bag 2    # Мешок II тира (4 слота)
/artifact bag 3    # Мешок III тира (6 слотов)
/artifact bag 4    # Мешок IV тира (8 слотов)
```

### Как использовать

1. Получите мешок командой `/artifact bag <тир>`
2. Откройте мешок Shift+ПКМ по нему
3. Положите артефакты внутрь
4. Положите мешок в хотбар
5. Все артефакты внутри будут работать!

### Ограничения

- В мешок можно класть **только артефакты**
- Количество слотов ограничено тиром мешка
- При заполнении — нельзя добавить больше
