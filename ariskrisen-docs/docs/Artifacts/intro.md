# Artifacts ✨

> Пассивные артефакты для Paper 1.21+

**Artifacts** — плагин добавляющий уникальные пассивные артефакты, которые работают просто находясь в инвентаре игрока.

## Основное

- [Установка](getting-started.md) — как установить плагин
- [Создание артефактов](creating-artifacts.md) — руководство по созданию
- [Эффекты](effects.md) — полный список эффектов
- [Настройки](configuration.md) — конфигурация плагина

## Команды

| Команда | Описание |
|---------|---------|
| `/artifact give <id> [игрок] [кол-во]` | Выдать артефакт |
| `/artifact list` | Список всех артефактов |
| `/artifact info <id>` | Информация об артефакте |
| `/artifact reload` | Перезагрузить конфиг |

## Требования

| Зависимость | Обязательно |
|------------|-------------|
| Paper/Spigot 1.21+ | ✅ Да |
| ItemsAdder | ❌ Нет (опционально для текстур) |
