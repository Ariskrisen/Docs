# Artifacts ✨

> Пассивные артефакты для Paper 1.21+

**Artifacts** — плагин добавляющий уникальные пассивные артефакты, которые работают просто находясь в инвентаре игрока.

## Основное

- [Установка](getting-started.md) — как установить плагин
- [Создание артефактов](creating-artifacts.md) — руководство по созданию
- [Эффекты](effects.md) — полный список эффектов
- [Настройки](configuration.md) — конфигурация плагина

## Команды

| Команда | Описание |
|---------|---------|
| `/artifact give <id> [игрок] [кол-во]` | Выдать артефакт |
| `/artifact list` | Список всех артефактов |
| `/artifact info <id>` | Информация об артефакте |
| `/artifact reload` | Перезагрузить конфиг |
| `/artifact bag <тир> [игрок]` | Выдать аномальный мешок (1-4) |

## Требования

| Зависимость | Обязательно |
|------------|-------------|
| Paper/Spigot 1.21+ | ✅ Да |
| ItemsAdder | ❌ Нет (опционально для текстур) |

## Аномальные Мешки

Аномальные мешки позволяют хранить артефакты внутри себя и использовать их эффекты.

### Тиры мешков

| Тир | Слотов | Описание |
|-----|--------|---------|
| I | 2 | Начальный мешок |
| II | 4 | Продвинутый мешок |
| III | 6 | Редкий мешок |
| IV | 8 | Легендарный мешок |

### Особенности

- Мешок работает как обычная сумка (BUNDLE)
- Внутрь можно класть **только артефакты**
- Автоматическая валидация — не-артефакты удаляются
- Все артефакты внутри работают пассивно

### Как использовать

1. Выдайте мешок: `/artifact bag 2`
2. Положите артефакты в мешок через Shift+ПКМ или перетаскивание
3. Мешок с артефактами работает в хотбаре
