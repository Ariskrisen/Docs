### Why DialogMenus?
Unlike classic Chest-GUI plugins, DialogMenus uses the native Minecraft system interface (Dialogs API). This provides:
- Smooth appearance animations.
- Native support for control elements (sliders, text fields).
- Perfect rendering on any screen resolution and FOV.
- Absolutely no "laggy" ghost items in the inventory.
