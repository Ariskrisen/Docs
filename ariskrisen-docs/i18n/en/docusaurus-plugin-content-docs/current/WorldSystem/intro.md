# WorldSystem 🌍

> [!IMPORTANT]
> **This is a fork!** WorldSystem-Melm is a modified and enhanced version of the original WorldSystem plugin.

**WorldSystem** is a powerful world management plugin where every player can have their own fully customizable world. It's the perfect solution for anarchy, survival, or creative servers where players need private space.

## ✨ Key Features
- **Personal Worlds** - Each player gets their own private world.
- **World Templates** - Choose from different generation types (Plain, Meadows, Desert, Snow).
- **GUI Management** - Easy-to-use interface for configuring permissions, mobs, TNT, and much more.
- **Optimization** - Support for async loading and integration with Chunky.
- **Economy** - Vault integration for paid world creation.

## 📥 Installation
1. Download the latest version from [GitHub Releases](https://github.com/Ariskrisen/WorldSystem-Melm/releases).
2. Place the JAR file into your `plugins/` folder.
3. Restart the server.
4. Configure templates and permissions in `config.yml`.

---

### Dependencies
- **WorldEdit** (required) — for region and world management.
- **Vault** (optional) — for paid world creation.
- **PlaceholderAPI** (optional) — for statistics output.
